# program-synthesis-project
Recreation of the Filament HDL for our EECS 700: Intro to Program Synthesis course.

## Building

To build the binary of the project:

```
make
```

For documentation:

```
make docs
```

To run tests:

```
make test
```

To clean up:

```
make clean
```

## Sources
